How to Install Light manager + for Maya

1. Place the entire light_manager_maya folder to your prefered location.
2. Open Maya and go to the Script Editor.
3. In the Python tab, paste the following code:

exec(open(r"path\to\light_manager_maya\lightManager.py").read())

Replace 'path\to\light_manager_maya' with the actual path where you placed the folder.

or:

Directly drag and drop the lightManager.py into an empty Python tab.


4. Run the script. The Light manager + window should appear.

Optional:
- Add the script to a Maya shelf for quick access, and set the "manager_icon.png" file included.

For usage instructions, see user_guide.txt.
