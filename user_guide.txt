User Guide for Light Manager +
================================

Overview:
---------
Light Manager + is a tool designed to help you manage, organize, and control lights within your Maya scenes efficiently. 
It provides a user-friendly interface for quick access to light attributes and batch operations.

Features:
---------
- List all lights in the current scene
- Select and delete lights any light in the current scene.
- Batch edit attribures: adjust intensity, color, and other custom attributes for all your lights at once.
- Scatter lights into selected objects, or create light rigs. Select multiple objects in the Maya outliner and click populate.
- Quick access to arnold utils.
- Quick material (standarSurface) creation, asignment and scene clean up.
- Save and backup lighting configuration states easily.

How to Use:
-----------
1. Launch the tool as described in installation.txt.
2. The main window will display a list of all lights in your scene.
3. Select a light to view and edit its properties.
4. Use the provided controls to adjust intensity, color, visibility, and other attributes.
5. To batch edit, select multiple lights and use the batch controls.
6. Use the refresh button if you add or remove lights in the scene while the tool is open.
